# 🎵 dizical 数据库表结构

## 数据库文件

- **dizi.db** — 课程 + 缴费 + 练习项目/分类/每日练习
- **dizical.db** — (当前为空，保留)

路径: `~/dev/dizical/data/`

---

## lessons 表 (dizi.db)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 主键 |
| date | DATE | 上课日期 (YYYY-MM-DD) |
| time | TIME | 上课时间 (HH:MM, 默认 17:15) |
| status | TEXT | scheduled / attended / cancelled |
| fee | INTEGER | 学费 (默认 600) |
| fee_paid | BOOLEAN | 是否已缴费 (0/1) |
| is_holiday_conflict | BOOLEAN | 是否与节假日冲突 (0/1) |
| notes | TEXT | 备注 (如节假日名称) |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

---

## payments 表 (dizi.db)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 主键 |
| payment_date | DATE | 缴费日期 |
| amount | INTEGER | 缴费金额 (元) |
| lesson_ids | TEXT | 覆盖的课程 ID (逗号分隔，空=按月) |
| payment_method | TEXT | 付款方式 (固定 '现金') |
| notes | TEXT | 备注 |
| created_at | DATETIME | 创建时间 |

---

## settings 表 (dizi.db)

| 字段 | 类型 | 说明 |
|------|------|------|
| key | TEXT PK | 配置键 |
| value | TEXT | 配置值 |
| updated_at | DATETIME | 更新时间 |

---

## practice_categories 表 (dizi.db)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 主键 |
| name | TEXT | 大科目名称 (如：基本功、演唱、分析、小节) |
| sort_order | INTEGER | 排序顺序 |
| created_at | DATETIME | 创建时间 |

**示例数据**:
```
id=1  name=演唱    sort_order=3
id=2  name=基本功  sort_order=1
id=3  name=分析    sort_order=2
id=4  name=小节    sort_order=5
id=5  name=句子    sort_order=6
id=6  name=曲子    sort_order=7
id=7  name=单音    sort_order=4
```

---

## practice_items 表 (dizi.db)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 主键 |
| name | TEXT | 小科目名称 (如：单吐练习、长音、回娘家) |
| is_active | BOOLEAN | 是否启用 (0/1) |
| category_id | INTEGER FK | 关联 practice_categories(id) |
| created_at | DATETIME | 创建时间 |

---

## daily_practices 表 (dizi.db)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 主键 |
| date | DATE | 练习日期 (YYYY-MM-DD, 唯一) |
| items | TEXT | 练习项目 JSON 数组 |
| total_minutes | INTEGER | 总时长 (分钟) |
| log | TEXT | 详细练习进展/备注 |
| created_at | DATETIME | 创建时间 |

**items 字段格式 (JSON)**:
```json
[
  {"item": "全按作低音re", "minutes": 2},
  {"item": "单吐练习", "minutes": 6},
  {"item": "采茶扑蝶", "minutes": 9},
  {"item": "回娘家", "minutes": 3},
  {"item": "唱茉莉花（二）", "minutes": 12},
  {"item": "基本功长音", "minutes": 8}
]
```

---

## weekly_assignments 表 (dizi.db)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 主键 |
| week_start_date | DATE | 周开始日期 (周一，UNIQUE) |
| items | TEXT | 每周作业 JSON 数组 |
| notes | TEXT | 备注 |
| created_at | DATETIME | 创建时间 |

**items 字段格式 (JSON)**:
```json
[
  {"item": "长音", "requirement": "多练低音"},
  {"item": "回娘家", "requirement": "练了1遍"},
  {"item": "唱茉莉花（二）", "requirement": "分析"}
]
```

---

## 索引

```sql
CREATE INDEX idx_lessons_date ON lessons(date);
CREATE INDEX idx_practices_date ON daily_practices(date);
CREATE INDEX idx_assignments_week ON weekly_assignments(week_start_date);
```

---

## ER 关系

```
practice_categories (1) ───< (N) practice_items
                               │
                               └───< (N) daily_practices_items (规划中)
                                       └───< daily_practices

lessons (1) ───< (N) payments
```
