# PRD: 练习页面布局重构

**基于 MRD:** `MRD-计时器模块重构-260511.md`
**目标文件:** `src/kid_app/templates/practice.html`
**状态:** 待开发

---

## 1. 背景与目标

**问题:** 当前练习页面（practice.html）元素过多、色块杂乱，用户视觉疲劳。

**目标:** 重构为清晰的**左-中-右三栏布局**，减少视觉噪音：

| 左栏（~25%） | 中栏（~45%） | 右栏（~30%） |
|---|---|---|
| 老师要求展示 | 计时器核心功能 | 额外练习时间 |

---

## 2. 新布局结构

### 2.1 整体框架

```
┌──────────────────────────────────────────────────────────────────────┐
│  [今日已练习: X 分钟]                              [实时时钟]         │
├────────────────┬─────────────────────────────┬──────────────────────┤
│   🎯 老师要求    │        ⏱️ 计时器             │   🌿 额外练习时间      │
│                │                             │                      │
│  (选中科目后    │   [  00:00  ]               │   +5 / +10 / +15 min │
│   显示练习要求) │                             │   +20 / +30 min      │
│                │   [5][10][15][20][30] min   │                      │
│  (无要求时      │                             │   [输入分钟 添加]      │
│   显示为空/隐藏)│   [开始]  [提前结束]          │                      │
│                │                             │   记录列表...         │
│                │   [快速录入面板]（隐藏）       │                      │
├────────────────┴─────────────────────────────┴──────────────────────┤
│                      选择练习项目（4列色块网格）                       │
├──────────────────────────────────────────────────────────────────────┤
│                      📋 今日练习记录                                   │
└──────────────────────────────────────────────────────────────────────┘
```

### 2.2 三栏容器 CSS

```css
.timer-layout {
  display: grid;
  grid-template-columns: 25% 45% 30%;
  gap: 12px;
  align-items: start;
}
.timer-layout .panel {
  background: var(--card);
  border-radius: var(--radius);
  padding: 14px 12px;
}
```

### 2.3 各栏内容

#### 左栏 — 老师要求（Panel 1）
- 标题：`🎯 老师要求`
- 选中带要求的练习科目时显示要求文本
- 无要求时：隐藏整个 Panel 或显示"无特殊要求"
- 样式：渐变背景 `#FFF9C4 → #FFE082`，边框 `#FFD54F`

#### 中栏 — 计时器（Panel 2）
- 大字体计时显示 `72px`，红色 `#FF6B6B`
- 时间按钮行：`5 / 10 / 15 / 20 / 30 分钟`
- 操作按钮：`开始` / `提前结束`
- 确认区域（计时完成后）：显示"练了 X 分钟！打卡完成"
- 快速录入面板（不计时时展开）：时间按钮 + 直接打卡

#### 右栏 — 额外练习时间（Panel 3）
- 标题：`🌿 额外练习时间`
- 快捷按钮：`+5 / +10 / +15 / +20 / +30 分钟`
- 自定义输入：`[___] 分钟 [添加]`
- 已添加记录列表（可删除）
- 样式：渐变背景 `#E8F5E9 → #C8E6C9`，边框 `#66BB6A`

## 3. 交互逻辑

### 3.1 选中练习科目

1. 点击科目色块 → 选中高亮
2. **左栏**：检查该科目是否在当前 assignment 中有要求，有则显示，无则隐藏 Panel
3. **中栏**：timer-card 显示（如果隐藏的话）
4. **右栏**：refreshExtraSection() 更新

### 3.2 计时器流程

1. 选择时间（默认10分钟）→ 点击开始
2. 计时中：计时器倒计时，页面离开弹窗保护
3. 计时结束：播放提示音 + 撒花动画，显示打卡确认
4. 点击打卡 → 提交记录 → 刷新今日记录

### 3.3 快速录入

1. 不启动计时器，直接选择时间 → 点击"直接打卡"
2. 记录直接存入，不触发计时器逻辑

### 3.4 额外练习时间

1. 选择时间按钮或自定义输入，添加到 extra-log
2. 可多条累积
3. 最终随计时打卡一起提交（或独立提交）
4. 每条可单独删除

## 4. 响应式策略

**iPad 竖屏（主要场景，768px 左右）：**

```css
@media (max-width: 640px) {
  .timer-layout {
    grid-template-columns: 1fr;  /* 单列堆叠 */
  }
}
```

- iPad 横屏 / 桌面：保持三栏
- iPad 竖屏 / 手机：单列堆叠
